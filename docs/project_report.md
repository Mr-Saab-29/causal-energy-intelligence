# Project Report

## Objective

Build a Causal Energy Intelligence Platform that supports electricity price forecasting, causal analysis, and carbon-aware workload shifting.

## Planned Milestones

1. Define data source contracts and database schema.
2. Implement ETL and feature engineering.
3. Train and evaluate baseline forecasting models.
4. Specify the causal DAG and treatment/outcome definitions.
5. Implement causal effect estimation and counterfactual simulation.
6. Build workload shifting optimization.
7. Expose results through FastAPI.
8. Add Docker, monitoring, and optional Kubernetes deployment.
